### Installation on pawsey  
1. Download:  
```  
   git clone https://github.com/tmplink/tmplink_uploader.git 
   cd tmplink_uploader 
```
2. Installation:
```
   curl -L -o pawsey.sh https://raw.githubusercontent.com/Underbigbridge/useful-tools/refs/heads/main/tmplink/pawsey.sh
   chmod +x pawsey.sh
   ./pawsey.sh
   source ~/.bashrc
```
### Useage
1. GUI version: 
```
   tmplink
```
1. Command version:
```
   tmplink-cli -file  yourfile  -upload-server "https://tmplink-upload-acc.vxtrans.com"
```
